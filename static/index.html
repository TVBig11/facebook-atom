Facebook Atom feeds

Want an Atom feed of your Facebook news feed, ie your friends' posts, that you can subscribe to and read in your favorite RSS feed reader?




You need to get your own access token because Facebook's v2.x API only lets apps see your own posts, not your friends'.

Here's how to get an access token:

  1. Open the Facebook Graph API Explorer.
  2. Click the Get Access Token button.
  3. Click Extended Permissions.
  4. Click the read_stream checkbox.
  5. Click the Get Access Token button.
  6. Copy the access token into the text box above.
